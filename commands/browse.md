# /browse - Explore All Developer Roles

Browse and discover all 71 available developer roles and specialized skill paths.

## Usage

```
/browse [category]
```

## Description

This command allows you to explore the complete catalog of developer roles organized by category:

- **All Available Roles**: 71+ complete learning paths
- **Category Navigation**: Browse by specialty
- **Role Details**: Quick overview of each path
- **Difficulty Levels**: Beginner to Expert
- **Learning Time**: Estimated duration
- **Prerequisites**: What you need to know first
- **Salaries & Job Market**: Career insights

## Browse Features

### Quick View
```
/browse frontend
```
Shows all frontend-related roles with quick descriptions.

### Detailed View
```
/browse react --details
```
Deep dive into React ecosystem, advanced patterns, and specializations.

### Category Filters
```
/browse --beginner
```
Show only beginner-friendly paths.

```
/browse --job-market high-demand
```
Show roles with best job market demand.

### Comparison Mode
```
/browse compare react angular vue
```
Side-by-side comparison of multiple frameworks.

## Advanced Browse Options

### Search by Skills
```
/browse --search "state management"
```
Find all roles that teach state management.

### Search by Job Title
```
/browse --job "senior developer"
```
Shows roles that lead to senior positions.

### New & Trending
```
/browse --trending
```
Discover newly added roles and trending technologies.

### Recommended for You
```
/browse --for-you
```
Get personalized role recommendations based on your interests and skills.
