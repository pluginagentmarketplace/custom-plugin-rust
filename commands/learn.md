# /learn - Learning Path Selection

Start your learning journey by selecting a developer role and getting a personalized learning path.

## Usage

```
/learn
```

## Description

This command guides you through selecting a specific developer role from the 71 available roadmaps and provides:

- **Recommended Learning Path**: Step-by-step progression
- **Time Estimates**: How long each phase takes
- **Key Milestones**: What to achieve at each stage
- **Project Ideas**: Real-world projects to practice
- **Resources**: Books, courses, tutorials
- **Assessment Criteria**: How to measure progress

## Example Interaction

When you run `/learn`, you'll be prompted to:

1. **Select Your Goal**
   - Frontend Development
   - Backend Development
   - Full-Stack Development
   - Mobile Development
   - Data Science & AI
   - DevOps & Cloud
   - And many more...

2. **Choose Your Experience Level**
   - Beginner: No prior experience
   - Intermediate: Some programming knowledge
   - Advanced: Ready to specialize

3. **Get Your Personalized Path**
   - Week 1-2: Fundamentals
   - Week 3-6: Core Concepts
   - Week 7-10: Advanced Topics
   - Week 11+: Specialization

## Supported Roles

**Frontend** (HTML, CSS, JavaScript, TypeScript, React, Next.js, Vue, Angular)
**Backend** (Node.js, GraphQL, Spring Boot, ASP.NET Core, PHP)
**Languages** (Python, Java, Go, Rust, C++, Kotlin, Bash)
**Data & AI** (ML, AI Engineering, Data Science, MLOps)
**Cloud & DevOps** (AWS, Kubernetes, Docker, Terraform, Linux)
**Databases** (PostgreSQL, SQL, Redis, MongoDB)
**Mobile** (iOS, Android, React Native, Flutter)
**Blockchain** (Smart Contracts, Web3, DeFi)
**And 50+ more specialized roles...**

## Related Commands

- `/roadmap` - View specific role roadmap
- `/assess` - Evaluate your knowledge
- `/browse` - Explore all available roles
