# /rust-learn Command

Start a personalized Rust learning session.

## Usage

```
/rust-learn [topic]
```

## Topics

- `basics` - Variables, types, functions
- `ownership` - Ownership, borrowing, lifetimes
- `structs` - Structs, enums, pattern matching
- `error-handling` - Result, Option, error patterns
- `traits` - Traits, generics, type system
- `async` - Async/await, Tokio
- `testing` - Unit tests, integration tests
- `cargo` - Build system, dependencies

## Example

```
/rust-learn ownership
```

This will:
1. Explain the concept with diagrams
2. Show code examples
3. Provide exercises
4. Check understanding
