# /rust-practice Command

Practice Rust programming with hands-on exercises.

## Usage

```
/rust-practice [level]
```

## Levels

- `beginner` - Basic syntax, ownership basics
- `intermediate` - Traits, generics, error handling
- `advanced` - Async, unsafe, macros

## Exercise Format

Each exercise includes:
1. Problem description
2. Starter code
3. Expected output
4. Hints (optional)
5. Solution explanation

## Example

```
/rust-practice beginner
```

## Topics Covered

### Beginner
- FizzBuzz
- Temperature converter
- String manipulation
- Vector operations

### Intermediate
- Generic data structures
- Custom error types
- Iterator chains
- Trait implementations

### Advanced
- Async web client
- Custom derive macros
- FFI wrappers
