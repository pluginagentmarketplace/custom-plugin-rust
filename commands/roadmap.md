# /roadmap - View Developer Roadmap

Display the detailed roadmap for a specific developer role with learning objectives, prerequisites, and progression path.

## Usage

```
/roadmap [role-name]
```

## Description

This command displays a comprehensive roadmap for your chosen developer role, including:

- **Fundamentals**: Core concepts you must master
- **Intermediate Skills**: Building on the basics
- **Advanced Topics**: Specialized knowledge
- **Project Milestones**: Hands-on experience
- **Timeline**: Expected learning duration
- **Resources**: Official docs, courses, books
- **Community**: Relevant communities and forums

## Examples

```
/roadmap react
```
Shows the React learning path covering components, hooks, state management, etc.

```
/roadmap backend
```
Shows the Backend development roadmap with Node.js, databases, APIs, etc.

```
/roadmap machine-learning
```
Shows ML fundamentals, algorithms, deep learning, and production ML.
