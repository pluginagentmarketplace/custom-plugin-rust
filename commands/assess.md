# /assess - Knowledge Assessment

Take a comprehensive assessment to evaluate your current knowledge level in a specific technology or role.

## Usage

```
/assess [topic]
```

## Description

This command provides interactive assessments to measure your understanding and knowledge in various areas:

- **Knowledge Level Detection**: Beginner, Intermediate, Advanced, Expert
- **Strengths Identification**: What you know well
- **Gaps Discovery**: Areas that need improvement
- **Personalized Recommendations**: What to learn next
- **Benchmark Comparison**: How you compare to others at your level

## Assessment Types

### Multiple Choice Questions
Quick evaluation of core concepts.

```
Q: What does REST stand for?
A) Representational State Transfer
B) Remote Execution Service Technology
C) Resource Exchange System Transfer
```

### Practical Coding Challenges
Hands-on problem solving:

```
Implement a function to find the two sum problem:
Given an array of integers and a target,
return the indices of the two numbers.
```

### Scenario-Based Questions
Real-world application:

```
You're building a scalable API that receives 10K requests/sec.
How would you handle caching? Database optimization? Rate limiting?
```

### Projects Assessment
Full portfolio evaluation with your completed projects.

## Example Assessments

```
/assess javascript
```
Evaluates your JavaScript fundamentals, ES6+, async patterns, and DOM manipulation.

```
/assess system-design
```
Tests your understanding of scalability, databases, caching, and distributed systems.

```
/assess react
```
Assesses React component design, hooks, state management, and performance optimization.
